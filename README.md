# SnapBuilds

SnapBuilds is a web application designed to boost your productivity by helping you master keyboard shortcuts and find the perfect computer for your needs.

## Features

- **Shortcut Catalog**: A comprehensive and searchable database of keyboard shortcuts for a wide range of applications and operating systems. Filter by OS or app to quickly find what you're looking for.
- **AI PC Recommender**: An intelligent assistant that recommends the best PC or laptop configuration based on your budget, intended use, and specific requirements.

## Getting Started

To run the application locally, you will need to have Node.js and npm installed.

1.  **Install dependencies**:
    ```bash
    npm install
    ```

2.  **Set up environment variables**:
    Create a `.env` file in the root of the project and add your Groq API key:
    ```
    GROQ_API_KEY=your_api_key_here
    ```

3.  **Run the development server**:
    ```bash
    npm run dev
    ```

The application will be available at `http://localhost:9002`.
